## Description: <br>
Comprehensive OpenClaw security audit that checks gateway binding, credential exposure, channel policies, tool sandboxing, network/IP leaks, and macOS system security. <br>

This skill is ready for commercial/non-commercial use. <br>

## Use Case: <br>
Developers and operators use this skill to run security checks against local OpenClaw installations, review findings, and choose relevant remediation scripts after configuration changes or security concerns. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill directs the agent to clone and run external audit scripts that inspect local OpenClaw and system configuration. <br>
Mitigation: Inspect or pin the referenced repository before running it, run read-only checks first, and review reports before sharing them. <br>
Risk: Fix scripts can change gateway, credential, permission, or channel settings if approved. <br>
Mitigation: Require explicit user confirmation for each fix and explain the proposed change before execution. <br>
Risk: Network checks may contact external services and expose IP-related metadata. <br>
Mitigation: Skip network/IP checks unless the user approves the tool's external-service prompt. <br>
